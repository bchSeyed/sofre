افزونه مستقل مدیریت منو و سفارشات رستوران برای وردپرس + ووکامرس.

== Description ==

سفره یک افزونه کامل برای رستوران‌هاست که بدون نیاز به قالب خاص کار می‌کند.

* منوی رستوران با شورتکد `[sofre_menu]`
* داشبورد مدیریت سفارشات
* ورود OTP با موبایل
* سبد خرید کشویی
* روش‌های تحویل: پیک، حضوری، صرف در رستوران

== Installation ==

1. فایل `sofre.zip` را از طریق افزونه‌ها > افزودن > بارگذاری نصب کنید.
2. افزونه **سفره** را فعال کنید (ووکامرس باید فعال باشد).
3. صفحه «منوی رستوران» به‌صورت خودکار ساخته می‌شود.

== Frequently Asked Questions ==

= آیا به قالب ریحون نیاز دارد؟ =

خیر. سفره کاملاً مستقل است و صفحه منو با قالب داخلی خود نمایش داده می‌شود.
